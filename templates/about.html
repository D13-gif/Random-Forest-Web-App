Tentang Aplikasi

Sistem Klasifikasi Sampah Berbasis Random Forest Algorithm

Tujuan Proyek

Aplikasi ini dikembangkan sebagai tugas praktikum Mata Kuliah Kecerdasan Buatan pada Bab 8: Random Forest Algorithm di Universitas Bale Bandung. Tujuan utamanya adalah menerapkan algoritma Random Forest untuk mengklasifikasikan gambar sampah ke dalam enam kategori dengan akurasi tinggi.

Metodologi

Ekstraksi Fitur (2.884 fitur)
  • Histogram Warna HSV: 32 bins per channel = 96 fitur
  • HOG: Histogram of Oriented Gradients = 1.764 fitur
  • Piksel Datar: Grayscale 32x32 = 1.024 fitur
Model

Random Forest Classifier dengan n_estimators=200 dan max_depth=20 pada dataset Garbage Classification.

Teknologi

Backend: Python, Flask, OpenCV, Scikit-learn

Frontend: Bootstrap 5, JavaScript

Deployment: Gunicorn untuk Heroku/Railway
